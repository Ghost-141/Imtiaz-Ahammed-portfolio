Favicon samples for the portfolio. To use one, update index.html:

  <link rel="icon" href="/favicons/<filename>.svg" type="image/svg+xml" />

Samples included (64x64 viewBox, rounded-square base):
- favicon-ia.svg       : "IA" monogram (violet→cyan gradient)
- favicon-camera.svg   : camera/lens motif (CV-friendly)
- favicon-spark.svg    : spark/star motif (GenAI-friendly)
- favicon-neurons.svg  : simple neural-network nodes and links
- favicon-hex-ai.svg   : hexagon with "AI" type

All use palette tuned to the site (violet/indigo/cyan). SVGs scale to 16×16, 32×32, 48×48, etc.
